If you are unzipping this on  a  non-DOS/Win system, you may need to
convert DOS' cr/lf pairs into your own line ending format.   If  you
used InfoUnZip, you can use the -a option.

Be  sure  and read at least the docs/index.txt file, which lists and
describes each of the  documutation  files.  You might actually care
about some of that information.
